Usage

Some basic tips on how to use DORGE.
  • You can search by gene symbol at the dropdown box on the left in "by Gene" page. Type in part of the gene or cancer type and it will give auto-completion suggestions.
  • You can then click on a row in the table in "DORGE prediction table" to bring up the associated information in in "by Gene" page.
  • The gene names are from HUGO.
  • You can download the entire dataset or subsets using the Download buttons.
  • The underlying code is available at Github with explanations of the associated data.

FAQs

Q: How can I contact the developers?

A: Please email JIE LYU.

Q: Why we focus on cancer drivers, OGs and TSGs?

A: TSGs and OGs can help improve the anti-cancer drug development, clinical therapy and personalized medicine.

Q: When I type in a gene or cancer, it isn't autocompleted. Why is that?

A: If a gene or cancer doesn't come up, it means that it isn't in the gene list included in DORGE. Check that the gene symbol is the standard (HUGO name.

Q: Is this website updated?

A: DORGE is updating regularly.

Q: Where can I get the raw data to use?

A: The data can be downloaded by clicking on the Download buttons in the app. It is licensed under CC0. Once the paper is finalised, we'd appreciate if you would cite it if you use this data.

Q: Where can I find the code used to construct this knowledge base?

A: All of the codes are available in our Github repository. The viewer is built using Shiny.

Q: How can I cite this work?

A: The preprint is available on bioRxiv. A peer-reviewed article will be out soon.

Q: Where can I report a bug?

A: Please email JIE LYU or create an issue on the DORGE GitHub repo.
